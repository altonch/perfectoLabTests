# Perfecto Lab Tests
These are Known-Good tests I use for the Perfecto Lab.

There are four series of tests in this suite. This will cover usage of:

* Appium 1
* Appium 2
* Selenium 3
* Selenium 4

These tests are provided As-Is with no warranty or support from the Perfecto team.

A user can work with these tests to check their connectivity to the Perfecto cloud lab.

While I do not foree any risks these tests may cause, use these tests at your own volition.
